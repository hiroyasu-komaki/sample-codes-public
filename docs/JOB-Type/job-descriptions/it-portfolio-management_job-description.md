# 募集要項

## 募集職種

### ITポートフォリオマネジメント職(エキスパートレベル)

<br>

## 現状の課題

### ITポートフォリオ管理における専門能力の不足
- 既存のIT戦略実行を進める中で、IT投資全体の可視化・最適化・価値評価に関する専門能力や深い知見の不足が課題
- 現場においても個別のIT投資判断は実施しているが、ポートフォリオ全体視点での戦略的意思決定のスピード感や投資効果最大化において専門性の補完が必要な状況
- 特に、IT投資ポートフォリオの戦略的バランス管理、投資優先順位付けの客観的評価基準、投資リスク・リターンの可視化、ポートフォリオ全体でのシナジー創出など、IT投資の戦略的価値最大化において、現状の実行力は十分ではない

### IT部門グローバル化におけるポートフォリオ統括力不足
- 海外組織との連携強化、グローバル共通のITポートフォリオ管理基盤の構築、統一された投資評価・優先順位付け標準の確立が必要
- 複数地域・事業にまたがるIT投資の統合管理、効果的な投資配分戦略など、グローバルレベルでのポートフォリオ統括力強化

### 全社横断的ポートフォリオ管理実行力の強化
- IT部門単独ではなく、経営企画部門、財務部門、各事業部門などと連携し、ITポートフォリオマネジメントのプロセス標準を通して各部門との協働による実行力強化
- 将来的には、ITポートフォリオマネジメントオフィス(ITPMO)を組織化することもあるが、そのための試金石として、その中核人材が必要

<br>

## 期待する役割・貢献

### IT投資ポートフォリオ戦略実行力の直接強化
- 当社の戦略実行において不足している専門能力を直接補完し、ITポートフォリオマネジメントを通じたIT投資の戦略的価値最大化とリスク最適化の実行力を強化していただきます
- 現場の取り組みに専門的な視座とスピード感を付加し、ポートフォリオ全体での投資目標達成の確実性を高めていただきます。

### グローバルITポートフォリオ管理基盤の強化
- 統一されたITポートフォリオ管理標準の確立、海外組織との効果的な投資統括体制構築など、グローバルITポートフォリオ管理基盤の強化をリードしていただきます。

### 組織ITポートフォリオ管理能力の向上
- ITポートフォリオマネジメントに関する専門知識・ノウハウの社内展開、標準プロセスの整備、メンバーのスキル向上支援など、組織全体のITポートフォリオ管理能力向上に貢献していただきます。

<br>

## 必須要件

### 実務経験
- ITポートフォリオマネジメント・IT投資管理業務経験5年以上
- 大規模IT投資ポートフォリオにおける戦略策定・最適化役割での実務経験(中規模以上の企業3社以上での経験、またはグループ会社含む複数事業単位での統括経験)
- IT投資評価・優先順位付け・資源配分の実務経験3年以上

### 専門知識・スキル
- ITポートフォリオマネジメント・投資ポートフォリオ最適化
- IT投資評価手法(NPV、IRR、ROI、TCO等)・価値測定
- ポートフォリオリスク管理・リスク分散戦略
- IT投資優先順位付け・資源配分最適化
- 企業IT・システム管理に関する基礎知識
- ITインフラ・システム基盤に関する基礎知識
- 経営企画・統制に関する基礎知識
- 財務・管理会計に関する基礎知識

- リスクマネジメント
  
### コミュニケーション能力
- 社内外のステークホルダーとの調整・投資判断説明能力
- 経営層へのポートフォリオ戦略報告・投資提案スキル
- 複数部門・事業部門との連携・投資調整経験

### 語学力
- 日本語ネイティブレベル(日本語検定1級または同等レベル)
- 英語フルエントレベル(TOEIC 900点以上またはCEFR C1レベル以上)
- ビジネス実務と投資管理実務のそれぞれのシーンにおいて、相手に合わせて専門用語と平易な表現を使い分けられる程度に両言語を習熟

### システム・ツール
- Microsoft Office系ビジネスツールに関する習熟(特にExcel上級レベル、PowerBI等のBIツール、Visio等の可視化ツール使用経験必須)

### その他
- ITポートフォリオマネジメント・IT投資戦略分野における包括的なプロフェッショナリズム(専門性、職業倫理、責任感、継続的学習への取り組み)
- ITポートフォリオマネジメントを通して企業に貢献し変革したいという意思、および業界や社会に対してポジティブに取り組む意欲
- 上記目標実現のための継続的な自己研鑽・情報収集・知識更新への具体的な取り組みを実践

# 業務レベル要件定義

## 業務レベルの評価軸

### 影響範囲による評価
- **組織**: 所属組織内 / 部門横断 / 全社的 / グループ会社含む
- **地域**: 国内のみ / 国内中心 / 国内外含む / グローバル統括

### 難易度による評価
- **業務複雑性**: 定型業務 / 半定型業務 / 非定型業務 / 創造的業務
- **意思決定レベル**: 実行レベル / 戦術レベル / 戦略レベル / 経営レベル
- **専門性要求度**: 基礎知識 / 応用知識 / 専門知識 / エキスパート知識

<br>

## レベル定義

| レベル | 組織 | 地域 | 業務複雑性 | 意思決定レベル | 専門性要求度 | 特徴 |
|--------|------|------|-----------|---------------|-------------|------|
| **ジュニアレベル** | 所属組織内 | 国内のみ | 定型業務 | 実行レベル | 基礎知識 | 既定の手順・基準に従った業務実行、上位者の指導下での遂行 |
| **中級レベル** | 所属組織内〜部門横断 | 国内のみ〜国内中心 | 半定型業務 | 実行レベル〜戦術レベル | 基礎知識〜応用知識 | 一定の裁量権を持った業務遂行、チーム指導、改善提案 |
| **エキスパートレベル** | 部門横断〜全社的 | 国内中心〜国内外含む | 非定型業務 | 戦術レベル〜戦略レベル | 専門知識 | 専門領域でのリーダーシップ、制度設計・仕組み構築、重要案件担当 |
| **シニアレベル** | 全社的〜グループ会社含む | 国内外含む〜グローバル統括 | 創造的業務 | 戦略レベル〜経営レベル | エキスパート知識 | 全社・グループレベルでの戦略策定・統括、経営層への直接報告・提案 |
