# 募集要項

## 募集職種

### レコードマネジャー（記録管理専門職）（エキスパートレベル）

<br>

## 現状の課題

### 記録管理の専門能力と戦略的視点の不足
- 既存の文書管理体制は主にファイリングシステムに依拠しており、グローバル・スタンダードの記録管理（Records Management）の概念や専門能力が組織内に不足している状況
- 記録管理が単なる文書整理業務として位置付けられ、説明責任、リスク管理、業務の証拠保全という戦略的価値が認識されていない
- 特に、業務分類体系の構築、記録のライフサイクル管理、メタデータ管理、処分権限の設定など、ISO15489等の国際標準に基づく記録管理の実行力が十分ではない

### デジタル化時代における記録管理基盤の脆弱性
- 紙記録中心の管理から電子記録管理への移行において、記録の真正性・信頼性・完全性・利用性を確保する専門的な仕組みが未整備
- 電子記録のメタデータ管理、長期保存戦略、マイグレーション計画など、デジタル環境における記録管理の専門知識が不足
- 記録システムの設計・導入において、記録管理の要求事項を適切に反映できる専門人材が不在

### コンプライアンスと説明責任における記録管理の脆弱性
- 法的義務の履行、業務活動の証拠保全、説明責任の遂行において、記録管理が果たすべき役割が組織内で明確化されていない
- 情報公開請求、監査、訴訟対応等において、必要な記録を迅速かつ確実に提供できる体制が整備されていない
- 記録の不適切な廃棄、隠蔽、改ざんを防止する統制機能が弱く、組織のレピュテーションリスクが高まっている

### 全社横断的記録管理実行力の強化
- 記録管理は単独部門の業務ではなく、法務部門、コンプライアンス部門、IT部門、各事業部門などと連携し、組織横断的な記録管理標準を確立する必要がある
- 将来的には、記録管理オフィス（Records Management Office）を組織化することも視野に入れ、その中核となる専門人材が必要

<br>

## 期待する役割・貢献

### 記録管理戦略の策定と実行力強化
- 当社において不足している記録管理の専門能力を直接補完し、グローバル・スタンダードに準拠した記録管理体制の構築と運用をリードしていただきます
- 記録管理を通じた説明責任の履行、リスク管理の強化、業務効率の向上に貢献し、組織の信頼性と透明性を高めていただきます

### デジタル記録管理基盤の構築
- 電子記録管理システムの要件定義、メタデータ枠組みの設計、記録の長期保存戦略の策定など、デジタル時代に対応した記録管理基盤の構築をリードしていただきます
- 紙記録と電子記録のハイブリッド環境における最適な管理手法を確立していただきます

### 組織の記録管理能力の向上
- 記録管理に関する専門知識・ノウハウの社内展開、記録管理方針・手順の整備、職員への教育研修の実施など、組織全体の記録管理能力向上に貢献していただきます
- 記録管理の重要性に関する組織文化の醸成をリードしていただきます

<br>

## 必須要件

### 実務経験
- 記録管理（Records Management）または文書管理業務経験5年以上
- 記録管理システムの導入・運用プロジェクトにおける中心的役割での実務経験（中規模以上のプロジェクト2件以上）
- 記録のライフサイクル管理（作成・取り込み・分類・保存・処分）の実務経験3年以上
- 業務分析に基づく分類体系または保存期間表の設計・運用経験

### 専門知識・スキル
- 記録管理の国際標準（ISO15489等）に関する深い理解
- 記録の信頼性要件（真正性・信頼性・完全性・利用性）に関する専門知識
- 記録のメタデータ管理・メタデータ枠組みの設計能力
- 業務分類体系（Business Classification Scheme）の構築手法
- 処分権限（Disposition Authority）の設定と管理
- 記録管理とリスク管理・コンプライアンスの関連性理解
- 電子記録管理・デジタルアーカイブに関する基礎知識
- 情報セキュリティ・アクセス制御に関する基礎知識
- 法規制環境（公文書管理法、情報公開法、個人情報保護法等）に関する基礎知識
  
### コミュニケーション能力
- 社内外のステークホルダーとの調整・説明能力
- 経営層への記録管理の価値・重要性に関する説明・提案スキル
- 各部門との連携・協働による記録管理標準の確立経験

### 語学力
- 日本語ネイティブレベル（日本語検定1級または同等レベル）
- 英語ビジネスレベル以上（TOEIC 800点以上またはCEFR B2レベル以上）
- 記録管理の専門用語を日英両言語で正確に理解し、使用できる能力

### システム・ツール
- Microsoft Office系ビジネスツールに関する習熟（特にExcel中級以上、SharePoint等の文書管理機能の理解）
- 文書管理システム・記録管理システムの利用経験、かつシステム導入における要件定義フェーズへの参加経験（記録管理要件の整理・必要機能の検討経験）
- メタデータ管理ツール、分類管理ツールの利用経験

### その他
- 記録管理分野における包括的なプロフェッショナリズム（専門性、職業倫理、守秘義務の徹底、継続的学習への取り組み）
- 記録管理を通じて組織の説明責任と透明性を高め、社会的信頼を向上させたいという強い意志
- 上記目標実現のための継続的な自己研鑽・情報収集・知識更新への具体的な取り組みを実践

<br>

- M&A・組織統合等における記録の移管・統合プロジェクト経験
- グローバル企業または複数拠点を持つ組織での記録管理標準化経験
- 情報公開請求対応の実務経験

### システム・ツール
- ECM（Enterprise Content Management）システムの導入・運用経験
- DAM（Digital Asset Management）システムの利用経験
- 電子署名・タイムスタンプ等の真正性担保技術の理解

### その他
- 記録管理分野での講演・執筆・教育活動経験
- ISO15489、ISO30300シリーズ等の国際標準策定・導入プロジェクトへの参画経験
- 業界または組織における記録管理ベストプラクティス事例の創出経験

# 業務レベル要件定義

## 業務レベルの評価軸

### 影響範囲による評価
- **組織**: 所属組織内 / 部門横断 / 全社的 / グループ会社含む
- **地域**: 国内のみ / 国内中心 / 国内外含む / グローバル統括

### 難易度による評価
- **業務複雑性**: 定型業務 / 半定型業務 / 非定型業務 / 創造的業務
- **意思決定レベル**: 実行レベル / 戦術レベル / 戦略レベル / 経営レベル
- **専門性要求度**: 基礎知識 / 応用知識 / 専門知識 / エキスパート知識

<br>
