# 募集要項

## 募集職種

### ビジネスアーキテクト職(エキスパートレベル)

<br>

## 現状の課題

### ビジネスアーキテクチャ設計における専門能力の不足
- 既存のIT戦略実行を進める中で、ビジネス戦略とIT実装を橋渡しする専門能力や深い知見の不足が課題
- 現場においても業務プロセス改善は実施しているが、エンタープライズ全体視点でのビジネスアーキテクチャ設計のスピード感や戦略的整合性において専門性の補完が必要な状況
- 特に、ビジネスケイパビリティマップの構築、ビジネスプロセスの標準化・最適化、バリューストリーム分析、ビジネス要件とIT要件の整合性確保など、ビジネスアーキテクチャを通じた事業価値最大化において、戦略目標の達成に向け現状の実行力は十分ではない

### IT部門グローバル化におけるビジネス統合力不足
- 海外組織との連携強化、グローバル共通のビジネスプロセス・ケイパビリティの構築、統一されたビジネスアーキテクチャ標準の確立が必要
- 複数地域・事業にまたがるビジネスアーキテクチャの統合設計、効果的な業務標準化・グローバル展開など、グローバルレベルでのビジネス統合力強化

### 全社横断的ビジネス変革実行力の強化
- IT部門単独ではなく、経営企画部門、財務部門、各事業部門などと連携し、ビジネスアーキテクチャのプロセス標準を通して各部門との協働による実行力強化
- 将来的には、ビジネスアーキテクチャ推進組織を組織化することもあるが、そのための試金石として、その中核人材が必要

<br>

## 期待する役割・貢献

### ビジネスアーキテクチャ構築・実行力の直接強化
- 当社の戦略実行において不足している専門能力を直接補完し、ビジネスアーキテクチャを通じた事業目標達成とビジネス・IT整合性の実行力を強化していただきます
- 現場の取り組みに専門的な視座とスピード感を付加し、ビジネス変革・デジタル変革の確実性を高めていただきます。

### グローバルビジネスアーキテクチャ基盤の強化
- 統一されたビジネスアーキテクチャ標準の確立、海外組織との効果的なビジネスプロセス統合体制構築など、グローバルビジネスアーキテクチャ基盤の強化をリードしていただきます。

### 組織ビジネスアーキテクチャ能力の向上
- ビジネスアーキテクチャに関する専門知識・ノウハウの社内展開、標準プロセスの整備、メンバーのスキル向上支援など、組織全体のビジネスアーキテクチャ能力向上に貢献していただきます。

<br>

- ビジネスアーキテクチャ設計・推進業務経験5年以上

# 業務レベル要件定義

## 業務レベルの評価軸

### 影響範囲による評価
- **組織**: 所属組織内 / 部門横断 / 全社的 / グループ会社含む
- **地域**: 国内のみ / 国内中心 / 国内外含む / グローバル統括

### 難易度による評価
- **業務複雑性**: 定型業務 / 半定型業務 / 非定型業務 / 創造的業務
- **意思決定レベル**: 実行レベル / 戦術レベル / 戦略レベル / 経営レベル
- **専門性要求度**: 基礎知識 / 応用知識 / 専門知識 / エキスパート知識

<br>

## レベル定義

| レベル | 組織 | 地域 | 業務複雑性 | 意思決定レベル | 専門性要求度 | 特徴 |
|--------|------|------|-----------|---------------|-------------|------|
| **ジュニアレベル** | 所属組織内 | 国内のみ | 定型業務 | 実行レベル | 基礎知識 | 既定の手順・基準に従った業務実行、上位者の指導下での遂行 |
| **中級レベル** | 所属組織内〜部門横断 | 国内のみ〜国内中心 | 半定型業務 | 実行レベル〜戦術レベル | 基礎知識〜応用知識 | 一定の裁量権を持った業務遂行、チーム指導、改善提案 |
| **エキスパートレベル** | 部門横断〜全社的 | 国内中心〜国内外含む | 非定型業務 | 戦術レベル〜戦略レベル | 専門知識 | 専門領域でのリーダーシップ、制度設計・仕組み構築、重要案件担当 |
| **シニアレベル** | 全社的〜グループ会社含む | 国内外含む〜グローバル統括 | 創造的業務 | 戦略レベル〜経営レベル | エキスパート知識 | 全社・グループレベルでの戦略策定・統括、経営層への直接報告・提案 |
