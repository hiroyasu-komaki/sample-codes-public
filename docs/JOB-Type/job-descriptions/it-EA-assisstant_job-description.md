# 募集要項

## 募集職種

### EA補佐職(中級レベル)

<br>

## 現状の課題

### EA活動推進における実務支援力の不足
- 既存のEA推進活動を進める中で、EAチームの円滑な運営やプロジェクト調整を支援する実務能力の不足が課題
- 現場においてもEA関連業務は実施しているが、EAリーダー・エンタープライズアーキテクトを効果的に補佐し、EA活動全体の実行精度とスピード感を高める実務支援力の補完が必要な状況
- 特に、EA成果物の作成・管理支援、ステークホルダー間の調整・連携促進、EA関連会議・ワークショップの運営支援、EAプロジェクトの進捗管理など、EA活動の実務推進において、円滑な運営を支える人材が必要

### EA活動の組織浸透における推進力不足
- EA活動を組織全体に浸透させるためには、各部門との日常的な連携、EA標準の普及促進、EA教育・啓発活動の実施など、現場レベルでの推進力が必要
- EAリーダー・エンタープライズアーキテクトが戦略的活動に専念できるよう、実務レベルでのEA推進を担う人材の確保

### 将来のEA専門人材育成基盤の構築
- 組織のEA能力を中長期的に向上させるためには、実務経験を通じてEA専門知識・スキルを習得し、将来的にEAリーダーやエンタープライズアーキテクトとして活躍できる人材の育成が必要
- EA補佐職として実務経験を積み、段階的にスキルアップしていく育成プログラムの確立

<br>

## 期待する役割・貢献

### EA活動の実務推進・運営支援
- EAチームの円滑な運営を支援し、EAリーダー・エンタープライズアーキテクトが戦略的活動に専念できる環境を整備していただきます
- EA関連プロジェクトの調整業務、成果物管理、会議運営など、EA活動の実務面を着実に推進していただきます。

### ステークホルダー連携・調整の促進
- 各部門・プロジェクト間の連携を円滑にし、EA活動に必要な情報収集・共有・調整を効果的に実施していただきます
- EA活動の組織浸透に向けた現場レベルでの推進・啓発活動を担っていただきます。

### EA専門人材としての成長
- 実務経験を通じてEAの専門知識・スキルを段階的に習得し、将来的にEAリーダー・エンタープライズアーキテクトとして活躍できる人材へと成長していただきます。

<br>

## 必須要件

### 実務経験
- IT業務・システム開発業務経験3年以上
- プロジェクト管理・調整業務の実務経験2年以上
- 複数部門との連携・調整業務の経験

### 専門知識・スキル
- プロジェクト管理・調整業務の基礎知識
- システム開発・IT業務に関する基礎知識
- ドキュメンテーション・資料作成能力
- 企業IT・システム管理に関する基礎知識
- エンタープライズアーキテクチャ(EA)に関する基礎知識(または学習意欲)
- ビジネスプロセスに関する基礎理解
  
### コミュニケーション能力
- 社内外のステークホルダーとの円滑なコミュニケーション能力
- 会議・ワークショップの運営・ファシリテーション能力
- 複数部門間の調整・橋渡し能力

### 語学力
- 日本語ビジネスレベル以上(日本語検定2級または同等レベル)
- 英語中級レベル以上(TOEIC 700点以上またはCEFR B2レベル以上)
- ビジネス文書の読み書き、基本的なビジネスコミュニケーションが可能なレベル

### システム・ツール
- Microsoft Office系ビジネスツールに関する習熟(Excel中級レベル、PowerPoint、Word、Visio等の基本操作)
- Microsoft系コラボレーションツールに関する習熟(SharePoint、Teams、OneNote、Planner等のチーム作業ツール群)
- プロジェクト管理ツール(Microsoft Project、Jira等)の基本的な利用経験

### その他
- EA活動推進・プロジェクト支援における基本的なプロフェッショナリズム(責任感、正確性、継続的学習への取り組み)
- EA活動を通して企業に貢献したいという意思、および業界や社会に対してポジティブに取り組む意欲
- EA専門人材としての成長を目指し、継続的な自己研鑽・情報収集・知識更新への具体的な取り組みを実践

# 業務レベル要件定義

## 業務レベルの評価軸

### 影響範囲による評価
- **組織**: 所属組織内 / 部門横断 / 全社的 / グループ会社含む
- **地域**: 国内のみ / 国内中心 / 国内外含む / グローバル統括

### 難易度による評価
- **業務複雑性**: 定型業務 / 半定型業務 / 非定型業務 / 創造的業務
- **意思決定レベル**: 実行レベル / 戦術レベル / 戦略レベル / 経営レベル
- **専門性要求度**: 基礎知識 / 応用知識 / 専門知識 / エキスパート知識

<br>

## レベル定義

| レベル | 組織 | 地域 | 業務複雑性 | 意思決定レベル | 専門性要求度 | 特徴 |
|--------|------|------|-----------|---------------|-------------|------|
| **ジュニアレベル** | 所属組織内 | 国内のみ | 定型業務 | 実行レベル | 基礎知識 | 既定の手順・基準に従った業務実行、上位者の指導下での遂行 |
| **中級レベル** | 所属組織内〜部門横断 | 国内のみ〜国内中心 | 半定型業務 | 実行レベル〜戦術レベル | 基礎知識〜応用知識 | 一定の裁量権を持った業務遂行、チーム指導、改善提案 |
| **エキスパートレベル** | 部門横断〜全社的 | 国内中心〜国内外含む | 非定型業務 | 戦術レベル〜戦略レベル | 専門知識 | 専門領域でのリーダーシップ、制度設計・仕組み構築、重要案件担当 |
| **シニアレベル** | 全社的〜グループ会社含む | 国内外含む〜グローバル統括 | 創造的業務 | 戦略レベル〜経営レベル | エキスパート知識 | 全社・グループレベルでの戦略策定・統括、経営層への直接報告・提案 |
