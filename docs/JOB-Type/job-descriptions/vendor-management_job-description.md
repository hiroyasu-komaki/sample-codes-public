# 募集要項

## 募集職種

### ベンダーマネジメント職（エキスパートレベル）

<br>

## 現状の課題

### 戦略実行における専門能力の不足
- 既存の戦略実行を進める中で、ベンダーマネジメントに関する専門能力や深い知見の不足が課題
- 現場においてもコスト意識を高く持ちながら取り組んでいるが、戦略実行のスピード感や視座において専門性の補完が必要な状況
- 特に、パートナー戦略とオペレーショナルコスト最適化において、戦略目標の達成に向け現状の実行力は十分ではない

### IT部門グローバル化における実行力不足
- 海外組織との連携強化、グローバル共通基盤の構築、統一されたベンダーマネジメント標準の確立が必要
- ベンダーに対する最適な交渉戦略、効果的な協働体制の構築など、グローバルレベルでの実行力強化

### 全社横断的実行力の強化
- IT部門単独ではなく、調達部門、法務部門、経営企画部門などと連携し、ベンダーマネジメントのプロセス標準を通して各部門との協働による実行力強化
- 将来的には、ベンダーマネジメントオフィス（VMO）を組織化することもあるが、そのための試金石として、その中核人材が必要

<br>

## 期待する役割・貢献

### 戦略実行力の直接強化
- 当社の戦略実行において不足している専門能力を直接補完し、ベンダーマネジメントを通じたパートナー戦略とオペレーショナルコスト最適化の実行力を強化していただきます
- 現場の取り組みに専門的な視座とスピード感を付加し、目標達成の確実性を高めていただきます。

### グローバル実行基盤の強化
- 統一されたベンダーマネジメント標準の確立、海外組織・ベンダーとの効果的な協働体制構築など、グローバル実行基盤の強化をリードしていただきます。

### 組織実行能力の向上
- ベンダーマネジメントに関する専門知識・ノウハウの社内展開、標準プロセスの整備、メンバーのスキル向上支援など、組織全体の実行能力向上に貢献していただきます。


<br>

## 必須要件

### 実務経験
- ベンダーマネジメント業務経験5年以上
- プロジェクトにおけるベンダーコントロール役割での実務経験（中規模以上のプロジェクト3件以上）
- 契約交渉・契約管理の実務経験3年以上

### 専門知識・スキル
- 調達・購買業務に関する専門知識
- 契約法務の基礎知識
- コスト分析・価格交渉スキル
- リスクマネジメント
  
### コミュニケーション能力
- 社内外のステークホルダーとの調整・交渉能力
- 経営層への報告・提案スキル
- チームマネジメント経験

### 語学力
- 日本語ネイティブレベル（日本語検定1級または同等レベル）
- 英語フルエントレベル（TOEIC 900点以上またはCEFR C1レベル以上）
- ビジネス実務と契約実務のそれぞれのシーンにおいて、相手に合わせて専門用語と平易な表現を使い分けられる程度に両言語を習熟

### システム・ツール
- Microsoft Office系ビジネスツールに関する習熟（日常業務での使用経験必須）
- Microsoft系コラボレーションツールに関する習熟（SharePoint、Teams、OneNote、Planner等のチーム作業ツール群）
- 調達管理システム（SAP、Oracle等）の利用経験、かつシステム導入における要件定義フェーズへの参加経験（ビジネス要件整理・必要機能の検討経験）

### その他
- 調達・ベンダーマネジメント分野における包括的なプロフェッショナリズム（専門性、職業倫理、責任感、継続的学習への取り組み）
- ベンダーマネジメントを通して企業に貢献し変革したいという意思、および業界や社会に対してポジティブに取り組む意欲
- 上記目標実現のための継続的な自己研鑽・情報収集・知識更新への具体的な取り組みを実践

<br>

## 尚可（あれば望ましい要件）

### システム・ツール
- 契約管理システムの利用経験および要件定義フェーズへの参加経験

### 専門資格・認定
- 調達・購買関連の専門資格（CPP、CPSM等）
- プロジェクトマネジメント関連資格（PMP、PMI等）
- 契約・法務関連資格（ビジネス法務検定、契約実務士等）

### 実務経験
- 海外ベンダーとの直接交渉・契約締結経験
- BPR・M&A・事業統合等におけるビジネス変革・統合プロジェクト経験

### その他
- ベンダーマネジメント分野での講演・執筆・教育活動経験
- 同業他社でのベストプラクティス事例創出経験

# 業務レベル要件定義

## 業務レベルの評価軸

### 影響範囲による評価
- **組織**: 所属組織内 / 部門横断 / 全社的 / グループ会社含む
- **地域**: 国内のみ / 国内中心 / 国内外含む / グローバル統括

### 難易度による評価
- **業務複雑性**: 定型業務 / 半定型業務 / 非定型業務 / 創造的業務
- **意思決定レベル**: 実行レベル / 戦術レベル / 戦略レベル / 経営レベル
- **専門性要求度**: 基礎知識 / 応用知識 / 専門知識 / エキスパート知識

<br>

## レベル定義

| レベル | 組織 | 地域 | 業務複雑性 | 意思決定レベル | 専門性要求度 | 特徴 |
|--------|------|------|-----------|---------------|-------------|------|
| **ジュニアレベル** | 所属組織内 | 国内のみ | 定型業務 | 実行レベル | 基礎知識 | 既定の手順・基準に従った業務実行、上位者の指導下での遂行 |
| **中級レベル** | 所属組織内〜部門横断 | 国内のみ〜国内中心 | 半定型業務 | 実行レベル〜戦術レベル | 基礎知識〜応用知識 | 一定の裁量権を持った業務遂行、チーム指導、改善提案 |
| **エキスパートレベル** | 部門横断〜全社的 | 国内中心〜国内外含む | 非定型業務 | 戦術レベル〜戦略レベル | 専門知識 | 専門領域でのリーダーシップ、制度設計・仕組み構築、重要案件担当 |
| **シニアレベル** | 全社的〜グループ会社含む | 国内外含む〜グローバル統括 | 創造的業務 | 戦略レベル〜経営レベル | エキスパート知識 | 全社・グループレベルでの戦略策定・統括、経営層への直接報告・提案 |
