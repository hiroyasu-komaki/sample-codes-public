# 募集要項

## 募集職種

### ITファイナンス職（エキスパートレベル）

<br>

## 現状の課題

### IT投資の価値可視化における専門能力の不足
- 既存のIT投資戦略実行を進める中で、ITコストとビジネス価値の関連性を定量的に評価・分析する専門能力や深い知見の不足が課題
- 現場においてもIT投資効果の測定に取り組んでいるが、戦略的意思決定のスピード感や経営層への説明責任において専門性の補完が必要な状況
- 特に、ITコスト配賦の自動化・リアルタイム可視化、業界標準との比較分析を通じたIT投資妥当性評価、統合的な予算管理プロセス、ITポートフォリオの価値評価・優先順位付けなど、IT投資ROIの最大化において、戦略目標の達成に向け現状の実行力は十分ではない

### IT部門グローバル化における財務管理実行力不足
- 海外組織との連携強化、グローバル共通のITコスト管理基盤の構築、統一されたIT財務管理標準の確立が必要
- IT投資に対する最適な予算配分戦略、効果的な投資効果測定など、グローバルレベルでの財務管理実行力強化

### 全社横断的財務実行力の強化
- IT部門単独ではなく、財務部門、経営企画部門、事業部門などと連携し、ITファイナンス管理のプロセス標準を通して各部門との協働による実行力強化
- 将来的には、ITファイナンス管理オフィス（ITFMO）を組織化することもあるが、そのための試金石として、その中核人材が必要

<br>

## 期待する役割・貢献

### IT投資戦略実行力の直接強化
- 当社の戦略実行において不足している専門能力を直接補完し、ITファイナンス管理を通じたIT投資の価値最大化とコスト最適化の実行力を強化していただきます
- 現場の取り組みに専門的な視座とスピード感を付加し、IT投資目標達成の確実性を高めていただきます。

### グローバルIT財務管理基盤の強化
- 統一されたITファイナンス管理標準の確立、海外組織との効果的なIT投資管理体制構築など、グローバルIT財務管理基盤の強化をリードしていただきます。

### 組織IT財務能力の向上
- ITファイナンス管理に関する専門知識・ノウハウの社内展開、標準プロセスの整備、メンバーのスキル向上支援など、組織全体のIT財務管理能力向上に貢献していただきます。

<br>

## 必須要件

### 実務経験
- ITファイナンス・IT投資管理業務経験5年以上
- 大規模IT投資プロジェクトにおける財務分析・投資効果測定役割での実務経験（中規模以上のプロジェクト3件以上）
- IT予算管理・コスト分析の実務経験3年以上

### 専門知識・スキル
- IT投資評価・価値測定
- ITコスト管理・配賦モデル
- IT予算管理・統制
- 財務・管理会計に関する専門知識
- 企業IT・システム管理に関する基礎知識
- ITインフラ・システム基盤に関する基礎知識
- 経営企画・統制に関する基礎知識
- リスクマネジメント
  
### コミュニケーション能力
- 社内外のステークホルダーとの調整・説明能力
- 経営層への財務報告・提案スキル
- 事業部門との連携・協働経験

### 語学力
- 日本語ネイティブレベル（日本語検定1級または同等レベル）
- 英語フルエントレベル（TOEIC 900点以上またはCEFR C1レベル以上）
- ビジネス実務と財務実務のそれぞれのシーンにおいて、相手に合わせて専門用語と平易な表現を使い分けられる程度に両言語を習熟

### システム・ツール
- Microsoft Office系ビジネスツールに関する習熟（特にExcel上級レベル、PowerBI等のBIツール使用経験必須）
- Microsoft系コラボレーションツールに関する習熟（SharePoint、Teams、OneNote、Planner等のチーム作業ツール群）
- ERP・財務管理システム（SAP、Oracle等）の利用経験、かつシステム導入における要件定義フェーズへの参加経験（財務要件整理・必要機能の検討経験）

### その他
- ITファイナンス・IT投資管理分野における包括的なプロフェッショナリズム（専門性、職業倫理、責任感、継続的学習への取り組み）
- ITファイナンス管理を通して企業に貢献し変革したいという意思、および業界や社会に対してポジティブに取り組む意欲
- 上記目標実現のための継続的な自己研鑽・情報収集・知識更新への具体的な取り組みを実践

<br>

- ITコスト配賦・可視化専門ツールの利用経験および要件定義フェーズへの参加経験

- プロジェクトマネジメント関連資格（PMP、PMI等）

### 実務経験
- 海外子会社のIT投資管理・財務統制経験
- M&A・事業統合等におけるITシステム統合・投資評価プロジェクト経験
- ITサービス事業のP&L管理・事業企画経験

### その他
- ITファイナンス分野での講演・執筆・教育活動経験
- 同業他社でのITコスト可視化・ポートフォリオ管理のベストプラクティス事例創出経験

# 業務レベル要件定義

## 業務レベルの評価軸

### 影響範囲による評価
- **組織**: 所属組織内 / 部門横断 / 全社的 / グループ会社含む
- **地域**: 国内のみ / 国内中心 / 国内外含む / グローバル統括

### 難易度による評価
- **業務複雑性**: 定型業務 / 半定型業務 / 非定型業務 / 創造的業務
- **意思決定レベル**: 実行レベル / 戦術レベル / 戦略レベル / 経営レベル
- **専門性要求度**: 基礎知識 / 応用知識 / 専門知識 / エキスパート知識

<br>

## レベル定義

| レベル | 組織 | 地域 | 業務複雑性 | 意思決定レベル | 専門性要求度 | 特徴 |
|--------|------|------|-----------|---------------|-------------|------|
| **ジュニアレベル** | 所属組織内 | 国内のみ | 定型業務 | 実行レベル | 基礎知識 | 既定の手順・基準に従った業務実行、上位者の指導下での遂行 |
| **中級レベル** | 所属組織内〜部門横断 | 国内のみ〜国内中心 | 半定型業務 | 実行レベル〜戦術レベル | 基礎知識〜応用知識 | 一定の裁量権を持った業務遂行、チーム指導、改善提案 |
| **エキスパートレベル** | 部門横断〜全社的 | 国内中心〜国内外含む | 非定型業務 | 戦術レベル〜戦略レベル | 専門知識 | 専門領域でのリーダーシップ、制度設計・仕組み構築、重要案件担当 |
| **シニアレベル** | 全社的〜グループ会社含む | 国内外含む〜グローバル統括 | 創造的業務 | 戦略レベル〜経営レベル | エキスパート知識 | 全社・グループレベルでの戦略策定・統括、経営層への直接報告・提案 |
