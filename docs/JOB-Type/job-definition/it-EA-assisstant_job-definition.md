# ジョブ定義フォーマット：EA補佐職（中級レベル）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| ジョブタイトル | EA補佐職（中級レベル） |
| 所属組織 | IT部門（EAチーム） |
| 職位レベル | 中級レベル |
| 雇用形態 | 正社員（ジョブ型） |

---

## 2. ジョブの目的（Job Purpose）
EA（エンタープライズアーキテクチャ）活動を実務的に支援し、  
EAチームの円滑な運営・調整・情報管理を通じてEA活動全体の精度とスピードを高める。  
将来的には、EA専門人材として成長し、EAリーダーやアーキテクトを補佐できる人材となることを目的とする。

---

## 3. 主な職務内容（Key Responsibilities）
- EA関連プロジェクトの進捗管理、調整業務、タスク追跡  
- EAチームの運営支援、会議・ワークショップの企画・運営補助  
- EA成果物（アーキテクチャドキュメント、図表等）の作成・更新支援  
- 各部門・ステークホルダーとの連携・情報共有促進  
- EA標準・テンプレートの維持・品質管理支援  
- EAツールやドキュメント管理システムの運用補助  
- EA関連データの収集・整理・報告支援  
- EA活動の社内広報・教育支援  

---

## 4. 成果基準（Expected Outcomes / KPIs）
- EA関連タスク・成果物の納期遵守率  
- 会議・報告資料の品質および正確性  
- ステークホルダー満足度（調整・情報共有面）  
- EA標準・ガイドライン遵守度  
- EA活動推進における効率性・生産性向上への寄与度  

---

## 5. 必要スキル・知識（Required Skills / Knowledge）
| カテゴリ | 内容 |
|-----------|------|
| 専門スキル | プロジェクト管理補助、ドキュメンテーション、調整・連携力 |
| 基礎知識 | IT業務、システム開発、企業IT・システム管理、EA概念 |
| ビジネススキル | コミュニケーション、ファシリテーション、資料作成能力 |
| ツールスキル | Microsoft 365（Teams、SharePoint、Planner）、Visio、Project、Jira |

---

## 6. 必要経験（Required Experience）
- IT業務・システム開発経験3年以上  
- プロジェクト管理・調整業務経験2年以上  
- 複数部門との調整・連携経験  
- EAまたは業務プロセス改善プロジェクト参画経験（尚可）  
- EAツールやモデリングツールの利用経験（尚可）

---

## 7. 権限と責任範囲（Authority & Accountability）
- EA活動における進捗・成果物管理の実務担当  
- 関係部門との調整・情報共有の実行責任  
- EAチーム運営（会議設定・資料管理・報告）の実務支援責任  
- EA標準準拠・品質維持の補助責任  

---

## 8. 関連ジョブ・連携関係（Interdependencies）
| 区分 | 関連ジョブ | 関係内容 |
|------|-------------|----------|
| 上位 | EAリーダー／エンタープライズアーキテクト | EA活動方針・標準策定の補佐 |
| 横 | IT企画担当、システム開発担当、プロジェクトマネージャー | プロジェクト・部門間の調整 |
| 下位 | EAサポートスタッフ、PMOアシスタント | タスク管理・資料作成支援 |

---

## 9. 成長パス（Career Path / Progression）
EA補佐職（中級）  
→ EAスペシャリスト（上級）  
→ エンタープライズアーキテクト／EAリーダー
