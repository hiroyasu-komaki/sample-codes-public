# ジョブ定義フォーマット：ITファイナンス職（エキスパートレベル）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| ジョブタイトル | ITファイナンス職（エキスパートレベル） |
| 所属組織 | IT部門（ITファイナンス管理 / ITFMO） |
| 職位レベル | エキスパートレベル |
| 雇用形態 | 正社員（ジョブ型） |

---

## 2. ジョブの目的（Job Purpose）
IT投資の価値最大化とコスト最適化を実現するため、  
IT投資評価、コスト管理、価値測定、財務統制を専門的にリードする。  
グローバル組織を含むIT財務基盤の整備・標準化を推進し、  
経営層の意思決定を支える財務透明性・統制力を強化する。

---

## 3. 主な職務内容（Key Responsibilities）
- IT投資戦略の策定支援および投資評価（ROI、NPV、IRR、TCO等）  
- ITコスト構造分析、配賦モデル設計、クラウド／オンプレコスト最適化  
- IT予算の策定・統制（予算編成、実績管理、差異分析）  
- IT投資ポートフォリオの価値評価・優先順位付け  
- IT財務ダッシュボードおよび経営報告資料の作成  
- グローバルITコスト管理および海外組織との協働推進  
- ERP・財務管理システム（SAP/Oracle）の要件定義・運用  
- 財務部門、経営企画、事業部門との横断的調整・統合プロセス整備  
- ITファイナンス標準プロセス・ガバナンスの構築・展開  

---

## 4. 成果基準（Expected Outcomes / KPIs）
- IT投資ROI・NPV改善率  
- コスト透明性の向上（ダッシュボード精度・更新頻度）  
- IT予算乖離の削減・予実精度向上  
- IT投資ポートフォリオ最適化指数（重複投資削減、優先順位最適化）  
- 財務報告の正確性・タイムリーさ（経営層満足度）  
- グローバルIT財務標準化の進捗・成熟度  

---

## 5. 必要スキル・知識（Required Skills / Knowledge）
| カテゴリ | 内容 |
|-----------|------|
| 財務・会計 | 財務会計、管理会計、投資評価、予算管理 |
| IT関連知識 | 企業IT、ITインフラ、システム管理、ITポートフォリオ |
| 分析スキル | コスト分析、データ分析、BIツール（PowerBI） |
| ビジネススキル | 経営層向け説明能力、調整力、ロジカルシンキング |
| リスク管理 | IT投資リスク、財務統制、コンプライアンス |

---

## 6. 必要経験（Required Experience）
- ITファイナンス・IT投資管理業務経験5年以上  
- 中規模以上のIT投資プロジェクトでの財務分析・効果測定実務経験  
- IT予算管理・コスト管理経験3年以上  
- ERP（SAP/Oracle等）での財務領域要件定義経験  
- 海外組織とのIT財務調整・協働経験（尚可）  
- M&A／PMIにおけるIT投資評価経験（尚可）  

---

## 7. 権限と責任範囲（Authority & Accountability）
- IT投資評価・IT財務分析のリード  
- IT予算の統制（レビュー・承認プロセス）  
- 財務部門と連携したITコスト配賦モデルの整備責任  
- 経営層向け財務報告および意思決定支援  
- グローバルIT財務管理標準の推進責任  

---

## 8. 関連ジョブ・連携関係（Interdependencies）
| 区分 | 関連ジョブ | 関係内容 |
|------|-------------|----------|
| 上位 | ITファイナンスマネージャー／CIO補佐 | 投資戦略・財務方針の承認 |
| 横 | 財務部門、経営企画部門、事業部門、調達部門 | 予算管理・財務統制・投資審査との連携 |
| 下位 | IT財務アナリスト、データアナリスト | データ分析・レポート支援 |

→ ITファイナンスマネージャー（ITFMO中核）  
→ シニアファイナンスリード／CFO補佐（IT領域財務統括）
