# 募集要項

## 募集職種

### IT人材育成スペシャリスト職（エキスパートレベル）

<br>

## 現状の課題

### IT人材育成における専門能力の不足
- 既存のIT戦略実行を進める中で、IT人材の計画的育成・スキル開発に関する専門能力や深い知見の不足が課題
- 現場においてもOJTや社内研修を実施しているが、戦略的意思決定のスピード感や人材育成の体系化において専門性の補完が必要な状況
- 特に、IT人材のスキルアセスメント・キャリアパス設計、効果的な育成プログラムの開発・運営、IT技術トレンドに対応した継続的学習体制の構築、組織的な知識継承・ナレッジマネジメントなど、IT人材の戦略的育成において、戦略目標の達成に向け現状の実行力は十分ではない

### IT部門グローバル化における人材育成実行力不足
- 海外組織との連携強化、グローバル共通のIT人材育成基盤の構築、統一されたIT人材開発標準の確立が必要
- IT人材のグローバル配置・ローテーション戦略、効果的な国際的人材交流など、グローバルレベルでの人材育成実行力強化

### 全社横断的人材育成実行力の強化
- IT部門単独ではなく、人事部門と連携しIT人材育成のプロセス標準を通した実行力強化
- IT部門内においてもシステム開発チーム、インフラ運用チームなど各ファンクションとの協働
- 将来的には、IT人材育成センター（IT Learning & Development Center）を組織化することもあるが、そのための試金石として、その中核人材が必要

<br>

## 期待する役割・貢献

### IT人材育成戦略実行力の直接強化
- 当社の戦略実行において不足している専門能力を直接補完し、IT人材育成を通じたIT組織の能力向上と人材価値最大化の実行力を強化していただきます
- 現場の取り組みに専門的な視座とスピード感を付加し、IT人材育成目標達成の確実性を高めていただきます。

### グローバルIT人材育成基盤の強化
- 統一されたIT人材育成標準の確立、海外組織との効果的なIT人材開発体制構築など、グローバルIT人材育成基盤の強化をリードしていただきます。

### 組織IT人材育成能力の向上
- IT人材育成に関する専門知識・ノウハウの社内展開、標準プロセスの整備、メンバーのスキル向上支援など、組織全体のIT人材育成能力向上に貢献していただきます。

<br>

## 必須要件

### 実務経験
- IT人材育成・人材開発業務経験5年以上
- IT分野における教育・研修プログラムの企画・運営実務経験（大規模プログラム3件以上）
- IT人材のスキルアセスメント・キャリア開発支援の実務経験3年以上

### 専門知識・スキル
- IT人材育成・開発手法
- ITトレーニング・カリキュラム設計
- eラーニング・デジタル学習環境構築
- 人材育成に関する人事領域に関する基礎知識
- システム開発・プロジェクト管理に関する基礎知識
- ITインフラ・システム運用に関する基礎知識
  
### コミュニケーション能力
- 社内外のステークホルダーとの調整・協働能力
- 経営層へのIT人材戦略報告・提案スキル
- IT技術者・非技術者双方との効果的なコミュニケーション経験

### 語学力
- 日本語ネイティブレベル（日本語検定1級または同等レベル）
- 英語フルエントレベル（TOEIC 900点以上またはCEFR C1レベル以上）
- ビジネス実務と教育実務のそれぞれのシーンにおいて、相手に合わせて専門用語と平易な表現を使い分けられる程度に両言語を習熟

### システム・ツール
- Microsoft Office系ビジネスツールに関する習熟（特にExcel上級レベル、PowerPoint上級レベル必須）
- Microsoft系コラボレーションツールに関する習熟（SharePoint、Teams、OneNote、Planner等のチーム作業ツール群）
- LMS（学習管理システム）・eラーニングプラットフォームの利用経験、かつシステム導入における要件定義フェーズへの参加経験（教育要件整理・必要機能の検討経験）

### その他
- IT人材育成・組織開発分野における包括的なプロフェッショナリズム（専門性、職業倫理、責任感、継続的学習への取り組み）
- IT人材育成を通して企業に貢献し変革したいという意思、および業界や社会に対してポジティブに取り組む意欲
- 上記目標実現のための継続的な自己研鑽・情報収集・知識更新への具体的な取り組みを実践

<br>

# 業務レベル要件定義

## 業務レベルの評価軸

### 影響範囲による評価
- **組織**: 所属組織内 / 部門横断 / 全社的 / グループ会社含む
- **地域**: 国内のみ / 国内中心 / 国内外含む / グローバル統括

### 難易度による評価
- **業務複雑性**: 定型業務 / 半定型業務 / 非定型業務 / 創造的業務
- **意思決定レベル**: 実行レベル / 戦術レベル / 戦略レベル / 経営レベル
- **専門性要求度**: 基礎知識 / 応用知識 / 専門知識 / エキスパート知識

<br>

## レベル定義

| レベル | 組織 | 地域 | 業務複雑性 | 意思決定レベル | 専門性要求度 | 特徴 |
|--------|------|------|-----------|---------------|-------------|------|
| **ジュニアレベル** | 所属組織内 | 国内のみ | 定型業務 | 実行レベル | 基礎知識 | 既定の手順・基準に従った業務実行、上位者の指導下での遂行 |
| **中級レベル** | 所属組織内〜部門横断 | 国内のみ〜国内中心 | 半定型業務 | 実行レベル〜戦術レベル | 基礎知識〜応用知識 | 一定の裁量権を持った業務遂行、チーム指導、改善提案 |
| **エキスパートレベル** | 部門横断〜全社的 | 国内中心〜国内外含む | 非定型業務 | 戦術レベル〜戦略レベル | 専門知識 | 専門領域でのリーダーシップ、制度設計・仕組み構築、重要案件担当 |
| **シニアレベル** | 全社的〜グループ会社含む | 国内外含む〜グローバル統括 | 創造的業務 | 戦略レベル〜経営レベル | エキスパート知識 | 全社・グループレベルでの戦略策定・統括、経営層への直接報告・提案 |
