# 募集要項

## 募集職種

### プログラムマネジメント職（エキスパートレベル）

<br>

## 現状の課題

### プログラムマネジメントにおける専門能力の不足
- 既存のIT戦略実行を進める中で、複数プロジェクト間の統合管理・依存関係管理に関する専門能力や深い知見の不足が課題
- 現場においても個別プロジェクトの管理は実施しているが、プログラム全体視点での戦略的意思決定のスピード感や実行精度において専門性の補完が必要な状況
- 特に、ポートフォリオ戦略とプロジェクト実行の橋渡し機能、プロジェクト間リスクの統合管理、プログラム全体でのリソース最適配分、ステークホルダー間の調整・コミュニケーション管理など、プログラムマネジメントを通じた価値実現において、戦略目標の達成に向け現状の実行力は十分ではない

### IT部門グローバル化におけるプログラム実行力不足
- 海外組織との連携強化、グローバル共通のプログラムマネジメント基盤の構築、統一されたプログラム管理標準の確立が必要
- 複数地域・組織にまたがるプログラムの統合管理、効果的な分散チーム運営など、グローバルレベルでのプログラム実行力強化

### 全社横断的プログラム実行力の強化
- IT部門単独ではなく、経営企画部門、財務部門、各事業部門などと連携し、プログラムマネジメントのプロセス標準を通して各部門との協働による実行力強化
- 将来的には、プログラムマネジメントオフィス（PMO）を組織化することもあるが、そのための試金石として、その中核人材が必要

<br>

## 期待する役割・貢献

### プログラム戦略実行力の直接強化
- 当社の戦略実行において不足している専門能力を直接補完し、プログラムマネジメントを通じたポートフォリオ戦略の実現とプロジェクト価値最大化の実行力を強化していただきます
- 現場の取り組みに専門的な視座とスピード感を付加し、プログラム目標達成の確実性を高めていただきます。

### グローバルプログラム管理基盤の強化
- 統一されたプログラムマネジメント標準の確立、海外組織との効果的なプログラム協働体制構築など、グローバルプログラム管理基盤の強化をリードしていただきます。

### 組織プログラム管理能力の向上
- プログラムマネジメントに関する専門知識・ノウハウの社内展開、標準プロセスの整備、メンバーのスキル向上支援など、組織全体のプログラム管理能力向上に貢献していただきます。

<br>

## 必須要件

### 実務経験
- プログラムマネジメント・複数プロジェクト統合管理業務経験5年以上
- 大規模ITプログラムにおけるプログラムマネージャー役割での実務経験（中規模以上のプログラム3件以上）
- プロジェクト間依存関係管理・リスク統合管理の実務経験3年以上

### 専門知識・スキル
- プログラムマネジメント・プロジェクトポートフォリオ管理
- プロジェクト間依存関係分析・管理
- プログラムリスク統合管理・制御
- ステークホルダーマネジメント・コミュニケーション戦略
- 企業IT・システム管理に関する基礎知識
- ITインフラ・システム基盤に関する基礎知識
- 経営企画・統制に関する基礎知識
- 財務・管理会計に関する基礎知識
- 事業運営・業務プロセスに関する基礎知識
- リスクマネジメント
  
### コミュニケーション能力
- 社内外のステークホルダーとの調整・統合管理能力
- 経営層へのプログラム進捗報告・提案スキル
- 複数部門・チーム間の協働・連携推進経験

### 語学力
- 日本語ネイティブレベル（日本語検定1級または同等レベル）
- 英語フルエントレベル（TOEIC 900点以上またはCEFR C1レベル以上）
- ビジネス実務とプログラム管理実務のそれぞれのシーンにおいて、相手に合わせて専門用語と平易な表現を使い分けられる程度に両言語を習熟

### システム・ツール
- Microsoft Office系ビジネスツールに関する習熟（特にExcel上級レベル、Project、PowerBI等のプロジェクト管理・分析ツール使用経験必須）
- Microsoft系コラボレーションツールに関する習熟（SharePoint、Teams、OneNote、Planner等のチーム作業ツール群）
- プログラム管理システム（Microsoft Project、Jira、Asana等）の利用経験、かつシステム導入における要件定義フェーズへの参加経験（プログラム管理要件整理・必要機能の検討経験）

### その他
- プログラムマネジメント・プロジェクト統合管理分野における包括的なプロフェッショナリズム（専門性、職業倫理、責任感、継続的学習への取り組み）
- プログラムマネジメントを通して企業に貢献し変革したいという意思、および業界や社会に対してポジティブに取り組む意欲
- 上記目標実現のための継続的な自己研鑽・情報収集・知識更新への具体的な取り組みを実践

# 業務レベル要件定義

## 業務レベルの評価軸

### 影響範囲による評価
- **組織**: 所属組織内 / 部門横断 / 全社的 / グループ会社含む
- **地域**: 国内のみ / 国内中心 / 国内外含む / グローバル統括

### 難易度による評価
- **業務複雑性**: 定型業務 / 半定型業務 / 非定型業務 / 創造的業務
- **意思決定レベル**: 実行レベル / 戦術レベル / 戦略レベル / 経営レベル
- **専門性要求度**: 基礎知識 / 応用知識 / 専門知識 / エキスパート知識

<br>

## レベル定義

| レベル | 組織 | 地域 | 業務複雑性 | 意思決定レベル | 専門性要求度 | 特徴 |
|--------|------|------|-----------|---------------|-------------|------|
| **ジュニアレベル** | 所属組織内 | 国内のみ | 定型業務 | 実行レベル | 基礎知識 | 既定の手順・基準に従った業務実行、上位者の指導下での遂行 |
| **中級レベル** | 所属組織内〜部門横断 | 国内のみ〜国内中心 | 半定型業務 | 実行レベル〜戦術レベル | 基礎知識〜応用知識 | 一定の裁量権を持った業務遂行、チーム指導、改善提案 |
| **エキスパートレベル** | 部門横断〜全社的 | 国内中心〜国内外含む | 非定型業務 | 戦術レベル〜戦略レベル | 専門知識 | 専門領域でのリーダーシップ、制度設計・仕組み構築、重要案件担当 |
| **シニアレベル** | 全社的〜グループ会社含む | 国内外含む〜グローバル統括 | 創造的業務 | 戦略レベル〜経営レベル | エキスパート知識 | 全社・グループレベルでの戦略策定・統括、経営層への直接報告・提案 |
