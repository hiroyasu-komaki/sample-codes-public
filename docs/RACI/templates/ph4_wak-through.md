#### 実施手順

**Step 1: オープニング（10分）**

```
【進行役スクリプト】
本日はウォークスルーにご参加いただきありがとうございます。

これから、実際の業務シナリオに沿って、時系列で
各タスクを追体験していきます。

皆さんには、シナリオ内の役割を演じていただきながら、
「この段階で本当に実行できるか?」
「必要な情報は揃っているか?」
「承認判断はできるか?」
などを具体的に確認していただきます。

疑問点や問題点があれば、その場でどんどん指摘してください。
それが貴重な改善の材料になります。
```

**説明事項**:
- [ ] ウォークスルーの進め方
- [ ] タイムボックス（各タスク5〜10分）
- [ ] 問題発見時の対応
- [ ] 役割分担の確認
