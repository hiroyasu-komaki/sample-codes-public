### 3.3 セッション詳細

#### セッション1: オープニング（10分）

**ファシリテータースクリプト**:
```
皆さん、本日はお集まりいただきありがとうございます。
本ワークショップの目的は、全員でRACIマトリクスを検証し、
実際に機能する体制を作り上げることです。

インタビューで皆さんからいただいた貴重な意見をもとに、
今日は集合的に課題を解決していきましょう。
```

**説明事項**:
- [ ] ワークショップの目的
- [ ] アジェンダと時間配分
- [ ] グラウンドルール（発言の尊重、時間厳守など）
- [ ] 期待される成果物

---

#### セッション2: 現状共有（30分）

**プレゼンテーション資料**:

**スライド1: RACIマトリクス全体像**
- 現在のRACIマトリクスを提示
- 主要なフェーズとタスクを説明

**スライド2: インタビュー結果サマリー**
- 実施状況（対象者数、完了率）
- 平均評価スコア
- 主要な発見事項

**スライド3: 共通課題トップ5**
- 最も多く指摘された課題を提示
- 各課題の影響度と緊急度

**質疑応答**: 5分
