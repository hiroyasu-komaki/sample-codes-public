## 1. 目的

本文書は、既存の文書管理規程における保存年限表を基盤として、デジタル記録を含む包括的な記録保持スケジュールに拡張し、ISO 30301に準拠した記録ライフサイクル管理を実現することを目的とする。

**統合の背景：**
- 既存の紙ベース文書管理規程・保存年限表の継承
- ISO 27001（ISMS）情報資産管理との統合
- 未管理のデジタル記録（メール、チャット等）の追加
- 紙と電子のハイブリッド管理への対応

---

## 2. 適用範囲

本スケジュールは、組織内で作成・受領されるすべての記録に適用される：
- **紙媒体記録**：従来の文書管理規程で管理されていた記録
- **電子媒体記録**：システム内データ、PDF、電子帳簿
- **ハイブリッド記録**：原本（紙）と副本（電子）の両方を保持
- **新規デジタル記録**：メール、チャット、ログ、SNS等の従来未管理記録

---

## 3. 用語定義

| 用語 | 定義 |
|------|------|
| 保持期間 | 記録を保存しなければならない最低期間 |
| 起算日 | 保持期間の計算開始日（作成日、完了日、終了日等） |
| 最終処分 | 保持期間満了後の記録の処理（廃棄、永久保存、アーカイブ等） |
| 評価選別 | 保持期間満了時の記録価値の再評価プロセス |
| リーガルホールド | 訴訟・監査等により廃棄を停止する措置 |
| **媒体区分** | **記録の保存形式（紙、電子、ハイブリッド）** |
| **原本** | **法的証拠力を持つ正式な記録（紙の場合は署名・捺印のある原本）** |
| **副本** | **原本のコピーまたはスキャンデータ（検索・参照用）** |
| **ハイブリッド管理** | **原本（紙）と副本（電子）を並行して保持する管理方式** |
| **未管理デジタル記録** | **従来の文書管理規程でカバーされていなかった電子記録** |
| **電子帳簿保存法対応** | **電帳法の要件（タイムスタンプ、検索機能等）を満たす電子保存** |

## 4. 既存文書管理からの移行対応表

### 4.1 従来の保存年限表からの変更点

| 従来項目 | ISO 30301拡張項目 | 追加理由 |
|---------|-----------------|---------|
| 文書種別 | 記録種別 + 媒体区分 | 紙/電子/ハイブリッドの明示 |
| 保管場所 | 物理保管場所 + 電子保管場所 | 二重管理の明確化 |
| - | 記録ID | 記録保持スケジュールとの紐付け |
| - | メタデータ要件 | 真正性・証拠能力の確保 |
| - | 起算日 | 保持期間の計算基準を明確化 |
| 廃棄方法 | 媒体別廃棄方法 | 紙と電子で異なる廃棄手段 |
| - | 業務プロセス | 記録発生源の明確化 |
| - | ISMS分類 | ISO 27001との連携 |

### 4.2 媒体区分の定義

| 媒体区分 | 説明 | 具体例 | 管理上の注意点 |
|---------|------|--------|--------------|
| **紙のみ** | 原本が紙媒体のみ | 従来の契約書原本、株主総会議事録 | 従来の文書管理規程を継承 |
| **電子のみ** | 完全電子化された記録 | 電子帳簿、システムログ、メール | 電帳法対応、バックアップ必須 |
| **ハイブリッド** | 原本（紙）+ 副本（電子）を並行管理 | 重要契約書、登記書類 | 両方の保持期間・廃棄を管理 |
| **移行中** | 紙→電子へ移行中（過渡期） | 給与台帳（2020年以前=紙、2021年以降=電子） | 切替時期を明記 |

### 6.1 紙媒体の廃棄方法

| 分類 | 廃棄方法 | 実施者 | 証跡 | 備考 |

### 6.2 電子媒体の廃棄方法

| 分類 | 廃棄方法 | 実施者 | 証跡 | 備考 |

| バックアップテープ | 消磁+物理破壊 | IT部門 | 破壊証明書 | データセンター処理 |

### 6.3 ハイブリッド管理の廃棄フロー

**原則：紙原本と電子副本の両方が保持期間を満了した場合に廃棄**

```
ステップ1: 保持期間満了確認
  ├─ 紙原本の保持期間チェック
  └─ 電子副本の保持期間チェック
     ↓
ステップ2: リーガルホールド確認（法務部）
     ↓
ステップ3: 廃棄承認（記録管理責任者）
     ↓
ステップ4: 並行廃棄実施
  ├─ 紙原本：溶解処理
  └─ 電子副本：論理削除
     ↓
ステップ5: 廃棄証跡記録
  ├─ 紙：溶解証明書
  └─ 電子：削除ログ
```

## 7. 保持期間の延長が必要なケース

以下の場合、通常の保持期間を超えて記録を保持する：

| ケース | 延長期間 | 承認者 | 記録方法 |

| 継続取引中の契約書 | 取引終了まで | 記録管理責任者 | 取引終了日を更新 |
| 歴史的価値のある記録 | 永久 | 管理責任者 | アーカイブ移管 |
| 調査・監査対応中 | 対応完了まで | 該当部門長 | 廃棄保留リストに登録 |

## 8. 廃棄プロセス

### 8.1 廃棄フロー（統合版）

```
1. 保持期間満了の自動検知（毎月1日実施）
   ↓
2. 廃棄候補リストの作成（媒体区分別）
   ↓
3. リーガルホールド確認（法務部）
   ↓
4. 廃棄承認申請（記録管理責任者）
   ↓
5. 廃棄承認（管理責任者）
   ↓
6. 媒体別廃棄実施（6.1-6.3参照）
   ↓
7. 廃棄証明書発行・記録
```

### 8.2 既存文書管理からの移行における注意事項

**紙文書の段階的電子化：**
- 新規作成文書：電子化を原則とする
- 既存紙文書：保持期間中はそのまま保管、満了時に廃棄
- 重要文書：スキャン後、原本も保管（ハイブリッド管理）

**電子化のタイミング：**
```
2024年1月～：新規契約書は電子署名+PDF保存を原則
2024年4月～：請求書の電帳法対応開始（スキャナ保存）
2025年1月～：稟議書の完全ワークフロー化
```

## 9. 評価選別プロセス

保持期間満了時に、以下の基準で記録の価値を再評価する：

### 9.1 評価基準

| 評価項目 | 内容 | 判断基準 |
|---------|------|---------|
| 法的価値 | 法的証拠・コンプライアンス | 訴訟リスク、法改正の可能性 |
| 業務価値 | 業務継続性 | 参照頻度、類似案件の発生可能性 |
| 歴史的価値 | 組織の記録・歴史 | 会社の節目、重要な意思決定 |
| 財務価値 | 資産評価 | 知的財産、ノウハウ |

### 9.2 評価結果と処理

| 評価結果 | 処理 | 承認者 |
|---------|------|--------|
| 高価値（永久保存） | アーカイブに移管 | 管理責任者 |
| 中価値（延長保存） | 保持期間を3-5年延長 | 記録管理責任者 |
| 低価値（廃棄） | 通常の廃棄プロセス | 部門長 |

## 10. 例外・特記事項

### 10.1 グローバル記録の取扱い

海外子会社・支店の記録は、現地法令と本スケジュールの**長い方**を適用する。

### 10.2 電子帳簿保存法対応

REC-F-001〜008は電子帳簿保存法の要件（タイムスタンプ、検索機能等）を満たすこと。

### 10.3 個人情報を含む記録

個人情報保護法に基づき、保持期間満了後は**速やかに廃棄**すること。

### 10.4 既存文書管理規程との関係

本スケジュールは、既存の文書管理規程における保存年限表を包含し、デジタル記録を追加したものである。紙文書については従来の管理方法を継承しつつ、電子化への移行を段階的に進める。

**【記入上の注意】**
- **既存の文書管理規程の保存年限表をベースに作成すること**
- **媒体区分（紙/電子/ハイブリッド/移行中）を必ず明記すること**

- 起算日は明確に定義すること（あいまいさを排除）
- 保持期間は法定期間以上とすること
- 個人情報を含む記録は慎重に取り扱うこと
- 年1回以上の定期レビューを実施すること
- **既存の部門文書管理責任者を記録管理責任者として活用すること**
