# 仮設検証のためのアンケート設計

## アンケートの基本設計

**目的**: チームタイプ別の課題仮説を定量・定性の両面から検証する

**回答方式**: 
- 5段階選択式（必須）: 5分以内で完了
- 自由記述式（任意）: 詳細な意見収集

**回答者情報**: 
- 所属チーム（必須）← **仮説検証の鍵**
- 回答は匿名化し、チームタイプ別に集計

<br>

#### アンケート本文

## 冒頭文

この度、半年前から開始した非同期オフライン会議体（OneNote活用）について、皆様のご意見をお聞かせください。いただいたフィードバックは、今後の運用改善に活用させていただきます。

- **所要時間**: 約5分（選択式のみの場合）
- **回答期限**: ○月○日（金）
- ※選択式は全て必須、自由記述は任意です

<br>
