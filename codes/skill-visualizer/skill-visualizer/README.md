# ITスキル人材開発 可視化プログラム

IT部門における組織開発のためのスキルインベントリー可視化ツールです。

## 必要要件

### システム要件
- Python 3.7以上
- 日本語フォントがインストールされた環境（macOS、Windows、Linux対応）

### 必須Pythonパッケージ
```
pandas>=1.3.0
matplotlib>=3.4.0
seaborn>=0.11.0
numpy>=1.21.0
```

<br>

## インストール

### 仮想環境の作成（推奨）
```bash
python3 -m venv venv

# Mac OSの場合: 
source venv/bin/activate

# Windowsの場合: 
venv\Scripts\activate
```

### 依存パッケージのインストール
```bash
pip install -r requirements.txt
```

<br>

## プロジェクト構成

```
.
├── main.py                          # エントリポイント
├── module/                          # 可視化モジュール
│   ├── role_radar_chart.py         # ロール別スキルレーダーチャート
│   ├── capability_skill_map.py     # ケイパビリティ別スキルマップ
│   └── skill_level_heatmap.py      # スキルレベル分布のヒートマップ
└── png/                             # 出力結果ディレクトリ
    ├── 1_role_radar_chart.png
    ├── 2_capability_skill_map.png
    └── 3_skill_level_heatmap.png
```

## 使い方

### 1. データファイルの配置

スキルインベントリーデータ（CSVファイル）を以下のパスに配置してください:
```
/mnt/user-data/uploads/converted_skill_data.csv
```

### 2. プログラムの実行

```bash
python3 main.py
```

実行すると、`png/`ディレクトリに3つの可視化結果が生成されます。

<br>

## 📊 生成される可視化

### 1. ロール別スキルレーダーチャート (`1_role_radar_chart.png`)
- **目的**: 各ロールの強み・弱みを一目で把握
- **内容**: 11ロール × 6カテゴリーのレーダーチャート
- **活用**: 組織全体のバランスと各ロールの特化領域を可視化

### 2. ケイパビリティ別スキルマップ (`2_capability_skill_map.png`)
- **目的**: 組織の4つのケイパビリティ充足度を評価
- **内容**: ヒートマップ形式で表示
  - リーダーシップケイパビリティ
  - ビジネスケイパビリティ
  - テクニカルケイパビリティ
  - デリバリケイパビリティ
- **活用**: どのケイパビリティが充足し、どこが弱いかを戦略的に把握

### 3. スキルレベル分布のヒートマップ (`3_skill_level_heatmap.png`)
- **目的**: 詳細なスキルギャップ分析
- **内容**: 42スキル項目 × 11ロールのマトリクス
- **活用**: 具体的にどのスキル×ロールの組み合わせが不足しているか特定

<br>

## データ形式

入力CSVファイルは以下の列を含む必要があります:

| 列名 | 説明 |
|------|------|
| カテゴリー | スキルの大分類 |
| サブカテゴリー | スキルの中分類 |
| スキル項目 | 具体的なスキル名 |
| ロール | 職種・役割 |
| 専門性 | 専門分野 |
| スキルレベル | a/b/c/d/z |
| スキルレベル_数値 | 5.0/3.0/2.0/1.0/2.5 |
| capability | ケイパビリティ種別 |

<br>

## 主な機能

- **自動集計**: ロール別、カテゴリー別、ケイパビリティ別の平均スキルレベルを自動計算
- **カラーコード**: スキルレベルを色で視覚的に表現（緑=高、赤=低）
- **統計情報出力**: コンソールに統計サマリーを表示
- **高解像度出力**: 300dpiのPNG形式で出力
