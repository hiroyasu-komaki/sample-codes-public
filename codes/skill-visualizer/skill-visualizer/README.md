# IT部門スキルアセスメント可視化ツール

IT部門のスキルアセスメントデータを多角的に可視化し、人材育成やスキルギャップ分析をサポートするPythonツールです。

## 概要

このツールは、IT部門の従業員のスキルデータをCSVファイルから読み込み、6種類の可視化グラフを生成します。各可視化は、スキルレベル、専門性、ロール、カテゴリー別の分析を行い、組織のスキルマッピングと人材育成計画の策定を支援します。

## 必要要件

### システム要件
- Python 3.7以上
- 日本語フォントがインストールされた環境（macOS、Windows、Linux対応）

### 必須Pythonパッケージ
```
pandas>=1.3.0
matplotlib>=3.4.0
seaborn>=0.11.0
numpy>=1.21.0
```

## インストール

### 仮想環境の作成（推奨）
```bash
python3 -m venv venv
source venv/bin/activate  # Windowsの場合: venv\Scripts\activate
```

### 依存パッケージのインストール
```bash
pip install -r requirements.txt
```

## 使い方

### 基本的な実行方法

1. CSVデータファイルを準備（詳細は[データフォーマット](#データフォーマット)を参照）
2. プログラムを実行
```bash
python3 main.py
```

3. 表示されるメニューから可視化を選択
```
================================================================
  IT部門スキルアセスメント可視化ツール
================================================================

以下の可視化を実行できます:

  1. ヒートマップ: ロール×スキルカテゴリー別の平均スキルレベル
  2. レーダーチャート: 専門性別のスキルプロファイル比較
  3. 積み上げ棒グラフ: スキルレベル分布
  4. 散布図: スキルギャップマトリックス
  5. 箱ひげ図: 専門性別のスキルレベル分布
  6. バブルチャート: 育成優先度マップ

  0. すべての可視化を実行
  q. 終了

================================================================
選択してください (0-6, q):
```

### 実行例

```bash
# すべての可視化を一括実行
選択してください (0-6, q): 0

# 特定の可視化のみ実行
選択してください (0-6, q): 1  # ヒートマップのみ
```

### 出力ファイル

生成されたグラフは`output/`ディレクトリに以下のファイル名で保存されます:

- `01_heatmap_role_category.png` - ヒートマップ
- `02_radar_chart_specialty.png` - レーダーチャート
- `04_scatter_skill_gap.png` - スキルギャップマトリックス

## データフォーマット

### CSVファイル仕様

デフォルトのCSVファイルパス: `csv/consolidated_skill_data.csv`

必須カラム:
- `ファイル名`: 人物を識別するID（同一ファイル名=同一人物）
- `ロール`: 役職・役割（例: プロダクトマネージャー、シニアエンジニア等）
- `専門性`: 専門分野（ビジネスアーキテクト、データサイエンティスト、ソフトウェアエンジニア、デザイナー）
- `カテゴリー`: スキルカテゴリー（ビジネス変革、データ活用、テクノロジー、デザイン、セキュリティ、パーソナルスキル）
- `スキルレベル`: スキルレベル（a, b, c, d, z）
  - a: 最高レベル（専門家）
  - b: 上級レベル
  - c: 中級レベル
  - d: 初級レベル
  - z: パーソナルスキル用（数値化対象外）
- `スキルレベル_数値`: スキルレベルの数値表現（a=5, b=4, c=3, d=2, z=1）

### CSVサンプル

```csv
ファイル名,ロール,専門性,カテゴリー,スキルレベル,スキルレベル_数値
person_001,プロダクトマネージャー,ビジネスアーキテクト,ビジネス変革,a,5
person_001,プロダクトマネージャー,ビジネスアーキテクト,テクノロジー,b,4
person_002,データアナリスト,データサイエンティスト,データ活用,a,5
person_002,データアナリスト,データサイエンティスト,ビジネス変革,c,3
```
